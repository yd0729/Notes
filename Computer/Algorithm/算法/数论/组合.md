# 组合

## 求组合数

$C_{n}^{m}=\cfrac{n!}{m!(n-m)!}=\cfrac{(m+1)(m+2)\cdots(n-1)n}{(n-m)!}$ ，所以 $m$ 越大，上边的分子就越小，越不容易溢出。且由定义可知 $C_n^m=C_n^{n-m}$ ，故当 $n-m>m$ 时应计算 $C_n^{n-m}$ 。

```cpp
long long comb(int n, int m) {
  if(m < n-m) m = n-m;
  long long ans = 1;
  for(int i = m+1; i <= n; i++) ans *= i;
  for(int i = 1; i <= n-m; i++) ans /= i;
  return ans;
}
```
