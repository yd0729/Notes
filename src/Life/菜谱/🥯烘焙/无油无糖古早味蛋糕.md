# 无油无糖古早味蛋糕

## 古早味

传统做法

## 配料表

|   材料   |          用量          |
| :------: | :--------------------: |
|   低粉   |          90g           |
|   鸡蛋   |          7 个          |
| 脱脂牛奶 |         60 ml          |
| 赤藓糖醇 |          60 g          |
|  香草精  |    几滴（用来去腥）    |
|  柠檬汁  | 几滴（用来稳定、去腥） |

## 做法

1. 蛋黄 + 低粉 + 香草精 + 牛奶，搅拌均匀

2. 蛋白 + 赤藓糖醇 + 柠檬汁，搅打至湿性发泡（尖钩会弯下去）

3. 用切拌的方式分三次将蛋白霜加入蛋黄糊中．

4. 加入模具后把表面刮平．

5. 在模具外加入 80 度热水．
